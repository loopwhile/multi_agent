# Skills for Codex CLI

이 파일은 Codex CLI에서 즉시 실행 가능한 업무 단위(Skill)를 정의한다.

## Available Skills

### scope-lock (Product Definition)
- Description: 요구사항을 분석하고 제품 범위를 고정한다.
- Instructions:
  - `docs/03_rules.md`
  - `tasks/scope_lock.md`
- Output: Scope Lock 문서

### architecture-design (System Design)
- Description: 기술 요구사항 기반 시스템 설계를 수행한다.
- Instructions:
  - `docs/03_rules.md`
  - `tasks/architecture_design.md`
- Output: 아키텍처 설계 문서

### implementation-plan (Development Strategy)
- Description: 구현 작업을 단계별로 분해하고 실행 계획을 수립한다.
- Instructions:
  - `docs/03_rules.md`
  - `tasks/implementation_plan.md`
- Output: 단계별 구현 계획서

### quality-check (Verification & QA)
- Description: 결과물이 Acceptance Criteria를 만족하는지 검증한다.
- Instructions:
  - `docs/03_rules.md`
  - `validation/quality_checklist.md`
- Output: Pass/Fail 검증 보고서

### feedback-analysis (Growth Analysis)
- Description: 운영 데이터/피드백을 분석해 개선 과제를 제안한다.
- Instructions:
  - `docs/03_rules.md`
  - `tasks/feedback_analysis.md`
- Output: 인사이트 요약 + 신규 Work Order 제안
