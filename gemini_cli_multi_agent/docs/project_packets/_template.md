## 사용법

이 파일은 에이전트가 프로젝트 컨텍스트를 주입받을 때 사용한다.  
역할 프롬프트(`roles/`)와 함께 이 파일을 로드하면,  
도구 독립적 역할 + 프로젝트 특수 맥락이 결합된다.

```
역할 프롬프트 (범용) + Project Packet (프로젝트 특수) = 실행 컨텍스트
```
